# طريقة بدون خطوات يدوية — خيارين:

## الخيار 1: رفع مباشر (أسرع)
أرسل لك ملف zip فيه:
- كامل الكود جاهز
- تعليمات بناء بسيطة (3 أوامر فقط)
- تبنيه على Mac محلي أو استخدام خدمة AppCircle

## الخيار 2: سكربت تلقائي
أكتب لك سكربت Python ترفعه على VPS ثاني (أو لابتوبك) ويعمل كل شي تلقائياً:
- يقرأ الشهادات
- يرفعها لـ GitHub API
- يشغل Build
- يرجع لك رابط OTA جاهز

## الخيار 3: Docker Container
أجهز Docker image فيه:
- macOS VM مع Xcode
- كل الإعدادات جاهزة
- تبني فيه مباشرة
